1. Buscar varios items bajo una condici√≥n

En esta lista hay varios objetos

const items = [
{name: 'Bike', price: 100},
{name: 'TV', price: 200}
];

Para saber si algun item del array obedece alguna condicion en especifico se puede utilizar:

const filteredItems = items.filter((item)=> {
return item.price <=100;
});

Para mostrar esta informacion en consola:

console.log(filteredItems)
