8 Must know Javascript array methods

By: Web dev simplified



1. Buscar varios items bajo una condición

En esta lista hay varios objetos

const items = [
{name: 'Bike', price: 100},
{name: 'TV', price: 200}
];

Para saber si algun item del array obedece alguna condicion en especifico se puede utilizar:

const filteredItems = items.filter((items)=> {
return items.price <=100;
});

Para mostrar esta informacion en consola:

console.log(filteredItems);



2. Map

Sirve para convertir un array completo, en uno mas simple con todos los valores de una clave específia.

const paises = [
{pais: 'España', capital: 'Madrid'},
{pais: 'E.E.U.U', capital: 'Washington'},
{pais: 'Alemania', capital: 'Berlin'}
];

const paisesCapital = paises.map((paises)=> {
return paises.capital;
});

Para mostrar esta informacion en consola:

console.log(paisesCapital)
